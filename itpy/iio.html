itpy.iio¶

This module contains methods for different I/O requirements.

itpy.iio.from_file(path_to_file, buffer=1)[source]¶

Make an iterable from the file that the path points to, the arguments mirrors how open() is implemented The optional buffering argument specifies the file’s desired buffer size: 0 means unbuffered, 1 means line buffered, any other positive value means use a buffer of (approximately) that size (in bytes).
